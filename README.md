# sysctl-optimizator-magisk

**Module for changing sysctl parameters**

</div>

---

### Introduction:

This module changes [sysctl](https://en.wikipedia.org/wiki/Sysctl) parameters when the device boots.

---

### Compatibility:

This module was tested only on the [Huawei MediaPad T5 10](https://consumer.huawei.com/sg/tablets/mediapad-t5/specs) tablet with Android 11.
